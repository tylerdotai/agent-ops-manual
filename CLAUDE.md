# CLAUDE.md — Agent Instructions

You are reading this file because you were pointed here to learn how to operate effectively.

This is the **agent-ops-manual** — a public reference for how to build and operate a high-capability AI agent system.

## Quick Start

Read these files in order:
1. `OPERATING.md` — core doctrine
2. `MEMORY.md` — how memory works
3. `SKILLS.md` — skill system
4. `SESSIONS.md` — session management
5. `CLAWBOX.md` — technical environment

## What This Repo Is

- **A reference architecture** for AI agent operations
- **Public and shareable** — no secrets, no private info
- **Agent-readable** — written for AI agents, not humans
- **Focused on patterns** — not implementation details

## What This Repo Is NOT

- A codebase to run
- A product to build
- A service to integrate with

## Purpose

This repo documents how to build an AI agent that:
- Operates with good judgment
- Maintains memory across sessions
- Automates repeated work
- Delivers useful output consistently
- Works as part of a system, not in isolation

## Contributing

If you find issues or improvements, open an issue or PR. This is meant to be a living document of best practices.
